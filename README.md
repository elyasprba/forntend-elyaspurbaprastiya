# React + TypeScript + Vite

## How To Install

- Clone this repo

```
  $ git clone https://github.com/elyasprba/forntend-elyaspurbaprastiya.git
```

- Go to folder repo

```
  $ cd frontend-elyaspurbaprastiya
```

- Install module

```
  $ npm install
```

- To start the project

```
  $ npm run start
```
